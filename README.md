# Game
Unzip spillet, og åbn start.bat for at spille spillet

Source koden for spillet ligger inde i .\src\main\java\dk\norakthes\game. For at få vist koden, enten download en text-editor, som [Notepad++](https://notepad-plus-plus.org/downloads/), eller vælg en fil og højre-klik og tryk på `Rediger`
